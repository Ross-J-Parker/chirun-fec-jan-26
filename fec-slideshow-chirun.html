What is Chirun?

  • Chirun is a tool that converts LaTeX to accessible web pages
  • Traditionally, lecture notes containing equations are created in LaTeX
  • To this day, standard LaTeX editor only output to PDF (a format designed for print)
  • PDFs are notoriously bad in terms of accessibility

What makes an equation accessible?

Accessible equations work for people and technologies, not just visually.

  • Perceivable ‚Äì not image-only
  • Operable ‚Äì navigable, not flattened

LaTeX Compatibility

Good practice:

  • Try to design for the web - not PDF
  • Use standard LaTeX packages wherever possible
  • If you‚Äôre struggling, use a template
  • To preserve PDF styling, use conditional logic (if, else).
